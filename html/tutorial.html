Get Started
To start using Open Maze software in your experiments you will need:

  1. The Unity Application
  2. Open Maze Software
  3. Sublime Text Editor

  1. Download and Install Unity

    2. Click Here to redirect to the Unity Download page. Please select which version of Unity you would like and begin the download process. Please note the the personal version of Unity is free but you must pass their terms and conditions to use it.

    After installing Unity, open up the application and login or create an account.

  2. Download and Install Open Maze

    3. Navigate to the Open Maze Github where you will download the latest version of Open Maze.

    Once you clicked on the link above, make sure that you are on the correct branch by ensuring that release is selected from the dropdown on the left-hand side of the page.

    Now, click the green Clone or Download button followed by Download as ZIP to begin downloading the Open Maze software to your computer.


    Once the ZIP file is done downloading, unzip it in an easy to access place on your computer and move on to the final step.

  3. Download and Install Sublime Text

    4. Please Note

    Navigate to https://www.sublimetext.com/3 to download a text editor. Select your operating system and then download the Text Editor.

  4. Run

    5. Now navigate to the main application and run it, after saving the configuration file you filled out in the previous step. Running the application is simple, just double click on it and voila

  5. Collect Data

    6. After running through the trial, navigate to this place in order to receive the output CSV file containing all the data you configured in the JSON file and more! You can do with this data as you like.
